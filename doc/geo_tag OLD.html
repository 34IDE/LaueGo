OLD Definition of geometry file for 34ID-E microdiffracation

by Jon Tischler, Aug 10, 2010

Used with old Roper detector (only one detector allowed)

An Example:

$dateWritten Thu, Jun 14, 2007
$timeWritten 21:46:16 (CDT)
$fileNote writen by 'writeGeoToFile.tcl' use this for June 13 too
$xalfd 0.15244 // CCD tilt (degrees)
$xbetd 0.17621
$dd 40.011 // height of CCD (micron)
$ddOffset 0.011 // dd = CCDy + ddOffset (micron)
$NxCCD 2084 // number of un-binned pixels in CCD, also called xdim, ydim
$NyCCD 2084
$dpsx 50.01 // size of CCD (mm)
$dpsy 50.168
$xcent 1085.1 // center of CCD in un-binned 1 based pixels
$ycent 923.9
$xbet 0.28893 // incident beam tilt (degrees)
$xgam 0.76751
$dia 52 // diameter of wire (micron)
$H0 -5889 // H of wire centered on the direct beam (micron)
$Hyc -5500.8 // H of wire centered on the vertical over the Si (micron)
$F 3100 // F of the wire for the Si calibration wire scan (micron)
$wireAxis {1,0,0} // direction of wire axis, should be close to (1,0,0)
